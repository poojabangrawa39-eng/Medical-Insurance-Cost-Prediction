# 🏥 Medical Insurance Cost Prediction

## 📌 Project Overview

Medical Insurance Cost Prediction is a Machine Learning project that predicts a person's medical insurance charges based on factors such as age, gender, BMI, number of children, smoking habits, and region.

The project uses a trained Machine Learning model to estimate insurance costs and provides an interactive web interface built with Streamlit for easy user interaction.

## Features

- Predict medical insurance charges instantly
- User-friendly Streamlit interface
- Machine Learning-based prediction
- Real-time cost estimation
- Simple and interactive design

## 📊 Dataset Features

The model uses the following input features:

- Age
- Gender
- BMI (Body Mass Index)
- Number of Children
- Smoker Status
- Region

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Medical-Insurance-Cost-Prediction.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the Streamlit application

```bash
streamlit run app.py
```
